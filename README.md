# My-Project1_System Archive Terminal v4.2

Key Features

1. Dynamic Category Management
- 사용자의 목표에 따라 체크리스트 카테고리를 자유롭게 추가/삭제할 수 있습니다.
- 각 카테고리별 진행 상태(Status)와 진척도(Progress)를 실시간으로 모니터링합니다.
2. Monthly Archive & Persistence
- 월별로 독립된 회고 데이터를 생성하고 관리합니다.
- LocalStorage를 활용하여 별도의 백엔드 없이도 브라우저 내에 안전하게 데이터를 유지합니다.
3. Strategy-Focused Retrospective (KPT)
- Keep: 유지해야 할 강점
- Problem: 개선이 필요한 병목 지점
- Try: 다음 스프린트를 위한 구체적 액션 플랜
4. Custom Visual Identity
- 직접 설계한 커스텀 SVG 로고를 통해 프로젝트의 고유한 정체성을 부여했습니다.
- 고효율 몰입을 위한 다크 모드 기반의 고대비 UI/UX.

🛠 Tech Stack
- Library: React.js
- Styling: Tailwind CSS
- Icons: Lucide-React
- State Management: React Hooks (useState, useEffect)
- Storage: Browser LocalStorage
